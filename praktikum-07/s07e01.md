## Aufgabe 2: Adjazenzliste vs Adjazenzmatrix

|Operation | AdjacencyMatrix | AdjacencyList |  
| :---        | :---: |   :---:   | 
| Space | O(N x N) | O(N + E) |
| Add Node | O(N x N) | ***O(1)*** |
| Remove Node | O(N x N) | ***O(N + E)*** |
| Add Edge | O(1) | O(1) |
| Remove Edge | ***O(1)*** | O(E) |
| Find existing Edge | ***O(1)*** | O(N) |
